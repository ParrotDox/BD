Работник (рекурсия)
1) В подчинении работника может быть множество рабочих (от 0 до многих)

Работник - бригада
1) Работник НЕ может существовать без бригады, работник может принадлежать к 1 бригаде (от 0 до 1)
2) Работник может управлять 1 бригадой, к работнику бригада может быть не назначена/назначена (от 0 до 1)
3) Бригада НЕ может существовать без работника, бригада может содержать множество работников (от 0 до до многих)
4) Бригадой может управлять работник, к бригаде работник-бригадир может быть не назначен/назначен (от 0 до 1)

Работник - Позиция
1) Работник не может существовать без позиции, работник имеет одну позицию (только 1)
2) Позиция может существовать без работника, одна позиция может быть у множества работников (от 0 до многих)

Работник - Свойство_Работник - Свойство
1) Работник может существовать без записей, работник может иметь множество записей (от 0 до многих)
2) Запись не может существовать без работника, каждая запись имеет одного рабочего (только 1)
3) Свойство может существовать без записей, свойство может иметь множество записей (от 0 до многих)
4) Запись не может существовать без свойства, каждая запись имеет одно свойство (только 1)

Актив (рекурсия)
1) На одном активе может находится множество активов (от 0 до многих)

Актив - Работник
1) Актив НЕ может существовать без работника-менеджера, работник-менеджер может быть не назначен/назначен (от 0 до 1)
2) Работник может существовать без актива, в управление работнику может быть назначен актив (от 0 до 1)

Актив - Бригада
1) Актив может существовать без бригады, на активе может быть несколько бригад (от 0 до многих)
2) Бригада НЕ может существовать без актива, бригада может быть привязана к 1 активу (от 0 до 1)

Т.О. - работник
1) Т.О. не может существовать без работника, Т.О. обязательно принадлежит 1 работник (только 1)
2) Работник может существовать без Т.О., работник может иметь множество Т.О. (от 0 до многих)

Т.О - автотранспорт
1) Т.О. не может существовать без автотранспорта, Т.О. обязательно принадлежит 1 автотранспорт (только 1)
2) Автотранспорт может существовать без Т.О., Автотранспорт может иметь множество Т.О. (от 0 до многих)

Т.О. - Т.О.-Деталь - Деталь
1) Т.О. может существовать без записей, Т.О. может иметь множество записей (от 0 до многих)
2) Запись не может существовать без Т.О., каждая запись имеет одно Т.О. (только 1)
3) Деталь может существовать без записей, деталь может иметь множество записей (от 0 до многих)
4) Запись не может существовать без детали, каждая запись имеет одну деталь (только 1)

Автотранспорт - Тип
1) Автотранспорт не может существовать без типа, каждый транспорт имеет один тип (только 1)
2) Тип может существовать без автотранспорта, один тип может быть у множества автотранспорта (от 0 до многих)

Автотранспорт - Статус
1) Автотранспорт не может существовать без Статуса, каждый транспорт имеет один Статус (только 1)
2) Статус может существовать без автотранспорта, один Статус может быть у множества автотранспорта (от 0 до многих)

Автотранспорт - Автотранспорт-Свойство - Свойство
1) Автотраснпорт может существовать без записей, автотранспорт может иметь множество записей (от 0 до многих)
2) Запись не может существовать без автотранспорта, каждая запись имеет один автотранспорт (только 1)
3) Свойство может существовать без записей, свойство может иметь множество записей (от 0 до многих)
4) Запись не может существовать без свойства, каждая запись имеет свойство (только 1)

Автотранспорт - транзит
1) Автотранспорт может сущестовать без транзитов, автотранспорт имеет множество транзитов (от 0 до многих)
2) Транзит не может сущестовать без транспорта, транзит имеет один автотранспорт (только 1)

Автотранспорт - маршрут
1) Автотранспорт НЕ может существовать без маршрута, к транспорту может быть не назначен/назначен маршрут (от 0 до 1)
2) Маршрут может существовать без автотранспорта, маршрут может иметь множество автотранспорта (от 0 до многих)

Маршрут - Тип
1) Маршрут не может существовать без типа, маршрут имеет одну позицию (только 1)
2) Тип может существовать без маршрута, тип может иметь множество маршрутов (от 0 до многих)
